## How To Use

1. Read principles first to align team boundaries and safety red lines.
2. Pick prompt templates by scenario (driver, A/V, WebRTC, porting, build, testing).
3. Enforce review and verification workflows before production use.

## Scope

- Embedded Linux / RTOS
- Audio/Video SDK (ffmpeg / rkmedia / MPP / WebRTC)
- Driver development and cross-platform porting
- Team-level AI-assisted development workflow
